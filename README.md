
cef-renderer
============

> NodeJS wrapper to run [renderer](https://github.com/ernestoalejo/renderer).


Installation
------------

You can simple do it by:

```bash
npm install cef-renderer --save-dev
```


API
---

### .karma

Type: `String`

Path to the karma binary of renderer.

### .seo

Type: `String`

Path to the seo binary of renderer.
